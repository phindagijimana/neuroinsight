## üéØ Testing Checklist

### Essential Tests (10 min)

- [ ] Downloaded correct build for OS
- [ ] Verified SHA256 checksum
- [ ] Installed successfully
- [ ] App launches
- [ ] Main window appears (3 tabs visible)
- [ ] Backend status: "Ready"
- [ ] No black/dark window
- [ ] Logs accessible (Ctrl+L / Cmd+L)
- [ ] No errors in error log

### Advanced Tests (30 min)

- [ ] Upload test file
- [ ] Processing starts
- [ ] Progress bar updates
- [ ] Processing completes
- [ ] Results viewable
- [ ] Viewer works (3 orientations)
- [ ] Stats tab shows data
- [ ] Memory usage reasonable
- [ ] No crashes during processing
